# Netflix-eda-analysis
Exploratory Data Analysis of Netflix Movies and TV Shows dataset.
Netflix Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on Netflix Movies and TV Shows dataset.

Files
- `EDA_Netflix_Debankita.ipynb` → Jupyter/Colab notebook with full EDA
- `netflix_titles.csv` → Dataset used for analysis

Libraries Used
- NumPy
- Pandas
- Matplotlib
- Seaborn

What’s Inside
- Cleaning and preprocessing the Netflix dataset
- Visualizing popular genres, ratings, durations, and trends
- Insights on Netflix content growth over years
